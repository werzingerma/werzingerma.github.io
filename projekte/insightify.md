  <h2>Insightify</h2>

  <p><strong>A command-line tool for dataset analysis with an optional interactive web dashboard.</strong></p>

  <p>
    <a href="https://github.com/werzingerma/insightify" class="primary-btn" target="_blank">View on GitHub</a>
  </p>

  <h3>Overview</h3>
  <p>
    Insightify streamlines dataset exploration by offering both CLI and web-based interfaces for performing comprehensive data analysis. Users can quickly assess data quality, generate visualizations, transform datasets, apply machine learning techniques, and export findings as professional reports—all from a single tool.
  </p>

  <h3>Key Features</h3>
  <ul>
    <li><strong>Multi-format Support</strong> – Works with CSV, Excel, JSON, and Parquet files</li>
    <li><strong>Quick Info</strong> – Instant dataset overviews with key statistics</li>
    <li><strong>Data Quality Assessment</strong> – Missing value detection and outlier identification</li>
    <li><strong>Visualization</strong> – Histograms, correlation plots, and scatter diagrams</li>
    <li><strong>Data Manipulation</strong> – Filtering, grouping, and aggregation operations</li>
    <li><strong>Machine Learning</strong> – Built-in anomaly identification capabilities</li>
    <li><strong>Report Generation</strong> – Export findings as HTML or PDF reports</li>
    <li><strong>Web Dashboard</strong> – Interactive Streamlit-based UI for visual exploration</li>
  </ul>

  <h3>Technology Stack</h3>
  <table>
    <tr><td><strong>Language</strong></td><td>Python (93%)</td></tr>
    <tr><td><strong>Web UI</strong></td><td>Streamlit</td></tr>
    <tr><td><strong>Data Processing</strong></td><td>Pandas, NumPy</td></tr>
    <tr><td><strong>Visualization</strong></td><td>Matplotlib, Plotly</td></tr>
  </table>

  <h3>Use Cases</h3>
  <p>
    Insightify is ideal for data scientists and analysts who need to quickly explore new datasets, identify data quality issues, or generate reports for stakeholders. The dual CLI/web interface makes it flexible for both scripted workflows and interactive exploration sessions.
  </p>
